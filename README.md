# Aditi Singh — Portfolio

**Live site:** [aditisingh.xyz](https://aditisingh.xyz)

This repository is the source for Aditi Singh’s portfolio site. It showcases her background in data science and machine learning, resume timeline, and hands-on project work.

## What You’ll Find on the Site

- **About & resume** — Education (UC Berkeley, Data Science major, CS minor), experience timeline, and a PDF resume link.
- **Projects** — Selected ML and data science work with short descriptions, tags, and dates. Several projects link to **interactive Jupyter notebooks** that open in the in-page viewer or can be downloaded.
- **Contact** — Contact form and direct contact details (email, phone, LinkedIn, GitHub) for reaching out.

## Tech Stack

The site is built with **plain HTML, CSS, and JavaScript** (no frameworks or build step) for fast load times and straightforward hosting. Project notebooks are served as static `.ipynb` files and rendered in the browser.

## Viewing the Portfolio

- **Online:** [https://aditisingh.xyz](https://aditisingh.xyz)
- **Resume:** Linked in the Resume section on the site, or open `Resume-AditiSingh.pdf` in this repo.
- **Projects & notebooks:** Project cards on the site link to the notebooks; the viewer works when the site is served over HTTP (e.g. on the live URL or a local server).
